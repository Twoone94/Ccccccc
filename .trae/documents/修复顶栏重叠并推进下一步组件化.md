- 解决首页与项目页顶栏中 `div` 与 `p` 元素重叠问题，确保三槽位布局不交叉
- 保留拖拽区域设置，保证桌面窗口可拖拽

## 修改方案
- 重构 `Toolbar` 为三槽位栅格布局，取消绝对定位居中
- 结构：左/中/右三列，居中列使用 `justify-center` 并支持文本截断
- 将现有页面使用方式保持不变（传入三个 children），避免大范围改动

## 具体改动
- 文件：`src/components/ui/Toolbar.tsx`
  - 用 CSS Grid 替换当前绝对居中结构：三列 `auto / 1fr / auto`
  - 中间槽位使用 `min-w-0` + `text-center` + `truncate`，避免与右侧溢出重叠
  - 示例实现：
    - 根容器：`grid grid-cols-[auto_1fr_auto] items-center border-b bg-app h-[48px] px-[16px]`
    - 左槽：`flex items-center gap-[8px]` + `no-drag`
    - 中槽：`flex items-center justify-center min-w-0` 渲染第二个 child
    - 右槽：`inline-flex items-center gap-[4px] h-[36px]` + `no-drag`
- 文件：`src/pages/SoftwareHomepage.tsx`
  - 顶栏中居中文本 `p` 添加 `text-center truncate`，去除不必要的 `pr-[16px]`
- 文件：`src/pages/ProjectInterface.tsx`
  - 顶栏保持一致用法，无需业务逻辑调整
